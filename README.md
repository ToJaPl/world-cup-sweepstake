# World Cup Sweepstake

A separate static site for tracking the sweepstake draw.

Open `index.html` in a browser. It pulls the 2026 FIFA World Cup scoreboard from ESPN's public scoreboard endpoint and calculates:

- Person leaderboard
- Team W-D-L
- Points
- Goal difference
- Upcoming fixtures
- Owner-vs-owner matchups

The draw is stored in `app.js`.
